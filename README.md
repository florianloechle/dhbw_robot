A semester project by the students of the business informatics (WI16B) course.
We are building a robot and control it through python and javascript!

## Tools used

- OpenCV
- Numpy
- Flask
- Click
- RP.GPIo
- YoloV3 Neural Network

## Install

Requires python >=3.6

```
pip3 install -r requirements.txt
```
